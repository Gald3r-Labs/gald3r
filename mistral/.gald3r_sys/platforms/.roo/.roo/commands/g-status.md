# gald3r Status

Produce a gald3r project status overview by reading `.gald3r/`:

1. Read `.gald3r/TASKS.md` and summarize counts by status indicator
   (`[📋]` pending, `[🔄]` in-progress, `[🔍]` awaiting verification, `[✅]` done, `[🚨]` needs attention).
2. Read `.gald3r/PLAN.md` for the current milestone/theme.
3. Read `.gald3r/BUGS.md` and report open vs. resolved bug counts.
4. Read `.gald3r/CONSTRAINTS.md` and list active constraints.
5. Surface anything stale or blocking.

Do NOT modify any `.gald3r/` file — this is a read-only status command.
