# g-skl-status

Report the current state of the gald3r project.

## Steps
1. Read `.gald3r/PROJECT.md` for mission and goals
2. Read `.gald3r/TASKS.md` for active tasks (status `[🔄]` and `[📋]`)
3. Read `.gald3r/BUGS.md` for open bugs
4. Summarize: mission, N active tasks, N open bugs, current plan focus
