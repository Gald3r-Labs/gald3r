# gald3r Framework

Access gald3r project management for this software project.

## Commands
- Check tasks: read `.gald3r/TASKS.md`
- Task detail: read `.gald3r/tasks/task{id}_{slug}.md`
- Log bug: append to `.gald3r/BUGS.md` + create `.gald3r/bugs/bug{id}_{slug}.md`
- Project mission: `.gald3r/PROJECT.md`

## Commit Format
```
type(scope): description

Task: #T{id}
```
