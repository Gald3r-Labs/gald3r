# g-skl-tasks

Manage gald3r task records.

## Operations
- **LIST**: Read `.gald3r/TASKS.md`, show all non-completed tasks
- **VIEW {id}**: Read `.gald3r/tasks/task{id}_*.md`
- **STATUS**: Count tasks by status emoji
