## How to use a platform template

1. Pick the folder matching your AI coding tool from the matrix below.

3. Restart your IDE / agent.

## Tier definitions

- **Tier 1 -- Fully Supported**: Tested by gald3r maintainers on every release. Bugs filed against this tier block releases.
- **Tier 2 -- Community Supported**: Tested by the community; contributions welcome.
- **Tier 3 -- Experimental**: Scaffold available; may have structural issues or missing features.
